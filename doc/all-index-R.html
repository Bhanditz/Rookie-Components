RadioButton — class, package com.gamerisker.controls
使用 RadioButton 组件可以强制用户只能从一组选项中选择一项。 该组件必须用于至少有两个 RadioButton 实例的组。 在任何给定的时刻，都只有一个组成员被选中。 选择组中的一个单选按钮将取消选择组内当前选定的单选按钮。 您可以设置 group 参数，以指示单选按钮属于哪个组。 皮肤设置继承ImageButton
