Rookie-Components
=================

什么是 Rookie Components ?
--------------------------
Rookie Components 是一套基于Starling的UI组件，主要借鉴flash component 、 feathers组件的代码。是一套简单的组件<br>
	对于游戏的一般应用够用。如果有任何需求请随便更改代码。<br>
    	如果你添加了优秀的组件请不要忘记还有很多同志们都需要。<br>
    	开源让世界更美好。为flash的移动开发付出自己的一份微薄力量。<br>
